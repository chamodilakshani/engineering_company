# Images folder

Drop your real image files here with these exact names and the site will pick
them up automatically (no code changes needed):

- `logo.png` — Meeya Engineering gear/logo mark, used in the navbar. Square,
  transparent background recommended (min 200x200px).
- `owner.jpg` — Portrait photo of Mihisara Gamage for the About section.
  Portrait orientation recommended (4:5 ratio, e.g. 800x1000px).

Until these files are added, the site shows clean placeholder icons in their
place, so nothing breaks.

You can also add more machinery/workshop photos here (e.g. `hero-bg.jpg`,
`gallery-1.jpg`) and reference them from the components if you'd like a photo
gallery added later.
